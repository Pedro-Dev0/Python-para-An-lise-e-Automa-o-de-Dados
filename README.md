# Python para Análise e Automação de Dados
Ambiente para uso no trabalho


🐍 Miniguia de Estudos Python: Do Zero ao Profissional com IA
Este repositório contém um projeto de curadoria e organização de conhecimento sobre a linguagem Python, desenvolvido como parte de um desafio prático na plataforma DIO. O objetivo é utilizar o NotebookLM como ferramenta de aprendizagem ativa para consolidar conceitos que vão desde a lógica básica até a arquitetura profissional.

🎯 Contexto e Objetivos
O mercado de tecnologia exige profissionais que não apenas saibam codificar, mas que compreendam a arquitetura e a automação de processos. Este projeto foca em:

Dominar a base: Consolidar conceitos de lógica e tipos de dados.

Automação: Entender como o Python pode otimizar tarefas repetitivas e manipulação de arquivos.

Qualidade de Código: Explorar a Programação Orientada a Objetos (POO) e testes para criar sistemas robustos.

Aprendizagem Ativa: Utilizar IA para estruturar roteiros de estudo personalizados.

📚 Curadoria de Fontes
Para alimentar o NotebookLM e gerar o guia final, foram selecionadas as seguintes fontes de alta autoridade:

Curso em Vídeo (Gustavo Guanabara): Referência em didática de lógica e Python básico no Brasil.

Automate the Boring Stuff with Python: Foco em aplicações práticas e automação de tarefas de infraestrutura e escritório.

FreeCodeCamp - Scientific Computing: Base teórica sólida com projetos práticos de algoritmos.

Documentação Oficial do Python (PT-BR): A fonte definitiva para consultas técnicas e novos recursos.

Real Python: Tutoriais aprofundados sobre boas práticas e ecossistema profissional.

🧠 Engenharia de Prompts e "Cicatrizes"
Para chegar ao resultado final, foram realizados testes iterativos com o NotebookLM:

Prompt 1 (Visão Geral): "Com base nas fontes enviadas, crie um roteiro de estudos de Python dividido por níveis de dificuldade."

Resultado: Ficou muito genérico, apenas listando tópicos.

Prompt 2 (Refinamento): "Reescreva o roteiro incluindo um desafio prático para cada etapa e explique o 'porquê' de usar cada comando mencionado."

Resultado: Melhorou muito, trazendo a parte didática e os desafios guiados.

Troubleshooting (Cicatrizes): Inicialmente, a IA misturou conceitos de Data Science com Web Development no nível iniciante. Foi necessário aplicar um prompt de restrição: "Mantenha o foco inicial em lógica pura e automação de arquivos, deixando frameworks específicos para a etapa final."
